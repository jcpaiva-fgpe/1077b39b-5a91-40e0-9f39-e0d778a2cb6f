Data extraction


Write a program that accepts in a loop some text from the user input with obligatory company e-mail addresses.

The program is supposed to pay attention only to text fragments where an e-mail address appeared, the rest of the text can be ignored. The program is to return the names separated by a period and numbers that appeared in the username of the email address. These numbers are to appear on the screen in descending order. If any of the numbers are repeated, it is to be included only once. Please pay attention to the correctness of the e-mail address. For simplicity, assume that alphanumeric and mandatory characters must appear in the corporate username. and @ (period and at sign). Then the name of the server followed by the TLD abbreviation.


An example of an execution flow:

    Computer: E-mail strings:

    User: To the manager address: dom123.budex@gmx.com

    Computer: 1) dom123.budex 3 2 1

    User: My friends are: 1Xi.1vg21@qq.cn, 10i.g12h1@qq.cn and 13i.c121a@qq.cn

    Computer: 1) 1Xi.1vg21 2 1

    Computer: 2) 10i.g12h1 2 1 0

    Computer: 3) 13i.c121a 3 2 1

    User: 1@.com

    Computer: False

    Computer terminates the program.

Note: If the user enters a sentence (from a new line) without a valid e-mail address, the computer has to display False and terminate the program.


To test how your code works, you can use the example above.

To complete the task, write your code and click the Submit button.
