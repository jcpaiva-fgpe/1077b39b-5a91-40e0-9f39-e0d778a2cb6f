Roman to Arabic


Write a program that converts each Roman number to an Arabic number. The user enters a Roman number as an input, but the output serves the Arabic number.

   I : 1   
   V : 5   
   X : 10  
   L : 50  
   C : 100 
   D : 500 
   M : 1000

Three different examples of an execution flow:

    Computer: Roman?

    User: MMXXI

    Computer: Arabic number: 2021


    Computer: Roman?

    User: LIV

    Computer: Arabic number: 54


    Computer: Roman?

    User: III

    Computer: Arabic number: 3
