Competition leader


Write a program that will read a name and the distance of each skijump from the user in the form Surname Distance , then update and determine the ledername and the best result in Ski Jumping Grand Prix, display it.


An example of an execution flow:

    Computer: Enter the result:
    User: Freund 159.6
    Computer: Leader: Freund 159.6
    Computer: Enter the result:
    User: Stoch 159.8
    Computer: Leader: Stoch 159.8
    Computer: Enter the result:
    User: Żyła 56.0
    Computer: Leader: Stoch 159.8
    Computer: Enter the result:
    User: Prevc 160.1
    Computer: Leader: Prevc 160.1
    Computer: Enter the result:
    User: (hits enter)
    Computer terminates the loop.

Note 1: The program is to terminate if the user does not enter any character or instead of entering the proper name and score format he enters for example only the word: end .
Note 2: Don't worry if two players get the same score (let's assume it's impossible).
