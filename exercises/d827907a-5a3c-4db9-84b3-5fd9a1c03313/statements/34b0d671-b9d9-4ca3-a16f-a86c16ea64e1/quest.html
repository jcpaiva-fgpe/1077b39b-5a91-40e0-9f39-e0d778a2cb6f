Leap year


Write a some code that reads a date as an input from the user in the format DD.MM.YYYY, the program will determine whether the given year is a leap year or not, please display a boolean value as an output.

First example of an execution flow:

    Computer: Date (DD.MM.YYYY)?

    User: 22.02.2000

    Computer: True

Second example of an execution flow:

Computer: Date (DD.MM.YYYY)?

User: 22.02.2001

Computer: False

If you would like to test how your code works, feel free to use the example above.
