Word with the highest number A


Write a program that will read any string from the user and as an output the word with the highest number of A-letters will be shown.


An example of an execution flow:

    Computer: String:

    User: Aha! This barbacane is voguish.

    Computer: barbacane

Note 1: Capital letters have the same importance as lower case, but be careful while displaying the word, the input should reflect the output.

Note 2: Don't worry if two words have the same letters (either display one of them or both).


If you would like to test how your code works, feel free to use the example above.

To complete the task, write your code and click the Submit button.


Good luck!
