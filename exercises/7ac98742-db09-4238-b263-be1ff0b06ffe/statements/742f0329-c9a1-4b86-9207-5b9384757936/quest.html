Write a program that reads the date from the user and based on a simple pattern, displays the temporary password to the user.

Date in the format DD.MM.YYYY will be transformed with the help of following formula: '[DD] * 2' + '[MM] * 2' + '[YYYY] * 2', that is: the day, month and year will be doubled. In Last but not least glue their individual values together and display the temporary password on the screen.


First example of an execution flow:

    Computer: Date (DD.MM.YYYY)?
    User: 21.07.1999
    Computer: Temporary password: 42143998

Second example of an execution flow:

    Computer: Date (DD.MM.YYYY)?
    User: 01.01.2001
    Computer: Temporary password: 224002

If you would like to test how your code works, feel free to use the example above.
