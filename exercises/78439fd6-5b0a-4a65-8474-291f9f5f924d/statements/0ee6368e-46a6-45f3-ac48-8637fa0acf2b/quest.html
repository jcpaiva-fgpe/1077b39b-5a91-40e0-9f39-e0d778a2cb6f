Word game


Write a program that allows the user to play words. The rules of this game are as follows:

  1. The computer displays the text ' Enter a word beginning with A: ' (User provides the word). If the User enters a word beginning with a different letter than A, he loses the game (an appropriate message is displayed, see the example of an execution flow below).
  2. The computer displays the text ' Enter a word starting with @: ' where the last letter of the previous word should appear instead of the @ sign. (User provides the word.)
  3. If the User does not enter the correct letter, they lose (an appropriate message 'Game over.' is displayed).
  4. If the User enters the correct letter, go back to point 2.
    5. The letters are not case-sensitive (a and A are the same).

An example of an execution flow:

    Computer: Word - A***?
    User: ADAPT
    Computer: Word - T***?
    User: TIK
    Computer: Word - K***?
    User: KIT
    Computer: Word - T***?
    User: No
    Computer: YGame over.

If you would like to test how your code works, feel free to use the example above.

To complete the task, write your code and click the Submit button.
