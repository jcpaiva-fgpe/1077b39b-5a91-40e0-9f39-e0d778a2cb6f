Add the digits in numbers together


Write a program that calculates the sum of the digits in the numbers entered by the user.

The program terminates if the user has not entered a positive integer, but has typed for example the string 'exit' in a new line.

First example of an execution flow:

    User: 727

    Computer: 16

    User: 10

    Computer: 1

    User: 3.0

    Computer terminates the loop

Second example of an execution flow:

    User: 0

    Computer: 0

    User: -0

    Computer terminates the loop

If you would like to test how your code works, feel free to use the example above.

To complete the task, write your code and click the Submit button.


Good luck!
