<h4> Converting temperature units </h4>

<p> Write some piece of code that converts temperatures from Kelvin to Celsius or from Celsius to Kelvin.
The program should initially read the choice from the user and as next should ask for temperatures as Kelvin or Celsius in a loop and converts it to Celsius or Kalvin degrees, apply the numeric rounding to 2-decimal place. if the user hasn't enter any correct number (in both cases: choice or temperature) the computer should terminate the program.</p>
