Text without four letters


Write a program that displays (in the first line) the inscription: 'String?' to the user, then the program reads some text from the user and displays the content without words containing exactly four letters from a new line. You can use the string split method.


An example of an execution flow:

    Computer: String?

    User: Lorem ipsum dolor sit amet consectetur adipiscing elit

    Computer: Lorem ipsum dolor sit consectetur adipiscing

If you would like to test how your code works, feel free to use the example above.

To complete the task, write your code and click the Submit button.
