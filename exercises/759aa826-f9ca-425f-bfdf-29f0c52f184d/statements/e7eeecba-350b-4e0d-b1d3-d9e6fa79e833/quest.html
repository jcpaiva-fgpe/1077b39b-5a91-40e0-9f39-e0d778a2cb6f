Palindromes


Write a program that checks if the word given by the user is a palindrome, ie from the back you read the same as from the front.


Examples of operation. The following form of writing (user) => (computer)

    Anna => True

    bulba => False

Letter case cannot be significant.

Note: Users can enter individual values from a new line (only).


If you would like to test how your code works, feel free to use the example above.

To complete the task, write your code and click the Submit button.


Good luck!
