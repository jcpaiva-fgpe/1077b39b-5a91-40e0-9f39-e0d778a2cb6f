Month of birth


Write a program that will read from the user the date of birth in the format: Day.Month.Year and displays the name of the month in which the user was born. Use the list: ['Jan', 'Feb', 'Mar', 'Apr', 'May', 'Jun', 'Jul', 'Aug', 'Sep', 'Oct', 'Nov', 'Dec'] in your program.

An example of an execution flow:

    Computer: Date (DD.MM.YYYY):
    User: 9.7.1986
    Computer: Jul

If you would like to test how your code works, feel free to use the example above.

To complete the task, write your code and click the Submit button.
