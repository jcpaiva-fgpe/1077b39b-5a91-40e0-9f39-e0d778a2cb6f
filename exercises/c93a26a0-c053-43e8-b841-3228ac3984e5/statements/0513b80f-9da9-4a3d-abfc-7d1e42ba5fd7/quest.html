Limits for shipping


Write a program that will calculate the total weight of packages to be shipped. An online retailer sells two products wholesale: rollers for painting and brushes.

Each roller weighs 75 grams. Each brush weighs 112 grams. One parcel can bear a maximum of 20 kg, so the total weight of one parcel cannot exceed 20 kg. If more than 20 kg of rollers / brushes, the number of packages must be adapted to the order. Write a program that reads with repetitions the number of rollers and the number of brushes of an order from the user. Then your program should calculate and display the total weight of the order, the number of packages, package type (below 10 kg - small package, over 10 kg large package. Please, refer to the el. from the parcels_id list). Finally, the program should display the weight of the lightest package.


parcels_id = ['small package', 'large package']


An example of an execution flow:

    Computer: Enter the number of rollers:

    User: 100

    Computer: Enter the number of brushes:

    User: 150

    Computer: The total weight of the packages is: 24300 g ~ 24.3 kg

    Computer: Number of packages: 2, the lightest: a small package, weight: 4.3 kg

    Computer: Next order?

    User: No.

    Computer terminates the program.

Attention: Only in case if the user enters 'yes' (any case variation) as an input, the program will proceed.

Apply the numeric rounding to 1-decimal place if the unit is set to kg.
