Limits for shipping


Write a program that will calculate the total weight of packages to be shipped. An online retailer sells two products wholesale: rollers for painting and brushes.

Each roller weighs 75 grams. Each brush weighs 112 grams. One parcel can carry a maximum of 20 kg, so the total weight of one parcel cannot exceed 20 kg. If more than 20 kg of rollers / brushes, the number of packages must be adapted to the order. Write a program that reads with repetitions the number of rollers (A) and the number of brushes (B) of an order from the user. Then your program should calculate and display the total weight of the order, the number of packages, package type (below 10 kg - small package [S], over 10 kg large package [L]. Please, refer to one of the items from the p_id list). Finally, the program should display the weight of the lightest package.


p_id = ['S', 'L']


An example of an execution flow:

    Computer: A?

    User: 100

    Computer: B?

    User: 150

    Computer: 24300g ~ 24.3kg

    Computer: #2 min: 4.3kg [S]

    Computer: Next?

    User: exit.

    Computer terminates the program.

Attention: Only in case if the user enters 'true' (any case variation) as an input, the program will proceed.

Apply the numeric rounding to 1-decimal place if the unit is set to kg (watch out, after the mass and the mass unit there is no space in-between).
